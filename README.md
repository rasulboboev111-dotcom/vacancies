# Vacancies HR System

Web-based HR System built with Laravel 11, PostgreSQL, InertiaJS, Vue 3, Vuetify 3, and Spatie libraries.

## Роли и доступы

| Роль | В интерфейсе | Сотрудники | Филиалы (справочник) |
|------|--------------|------------|----------------------|
| **Admin** | Админ | Все филиалы: полный доступ | Полный доступ |
| **User** | Пользователь (ответственный за филиал) | **Просмотр** всех филиалов; **добавление и изменение** только в своём `branch_id` | Просмотр всех; управление только админом |

Пользователь без привязки к филиалу (`branch_id`) не видит данные, пока админ не назначит филиал.

После изменения прав: `php artisan db:seed --class=RoleAndPermissionSeeder`
